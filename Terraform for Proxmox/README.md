Манифест создает заданное количество виртуальных машин из подготовленного cloud-init шаблона с переопределяемыми параметрами, уже имеющегося в Proxmox, назначает ip-адреса из заданного диапазона, создает пользователей внутри виртуальных машин, копирует созданный заранее и положенный в папку с этими файлами публичный SSH-ключ.
Для использования манифестов-склонировать репозиторий, подставить свои значения внутрь файлов, положить в папку с файлами публичный SSH-ключ
